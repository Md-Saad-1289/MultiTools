### MultiTools

    20+ Free Online Tools
    Responsive Design

    Fast & Lightweight

    No API Keys
    No Backend Required

🛠️ Built With

    HTML5
    CSS3
    Vanilla JavaScript

🚀 Run Locally

>Simply open "index.html" in your browser.
